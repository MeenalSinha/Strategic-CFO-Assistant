## 🚀 Quick Start

1. **Extract the ZIP file**
2. **Navigate to folder:** `cd strategic-cfo-assistant`
3. **Install:** `pip install -r requirements.txt`
4. **Run:** `streamlit run cfo_assistant.py`
5. **Open:** http://localhost:8501
