# 🚀 START HERE - Strategic CFO Assistant

Welcome to the Strategic CFO Assistant! This guide will get you started in 2 minutes.

---

## ⚡ Quick Start (3 Steps)

### Step 1: Install Dependencies (30 seconds)
```bash
pip install -r requirements.txt
```

### Step 2: Run the Application (10 seconds)
```bash
streamlit run cfo_assistant.py
```

### Step 3: Open in Browser
The app will automatically open at: http://localhost:8501

**That's it!** 🎉

## 🔧 Troubleshooting

**Port already in use?**
```bash
streamlit run cfo_assistant.py --server.port 8502
```

**Module not found?**
```bash
pip install --upgrade pip
pip install -r requirements.txt
```

**Data file not found?**
- Check that `data/upi_transactions_2024.csv` exists
- File should be 29MB with 250,001 rows
